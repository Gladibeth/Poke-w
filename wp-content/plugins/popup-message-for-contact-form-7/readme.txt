== Description ==

CF7 Lightbox Popup Success is offering features like Add Success message in to Fronted side in contact form 7. 
In wordpress admin there is all setting to popup to customize background color, font size and font color.

FEATURES
-Create popup for any Contact Form 7 form.
-Use different popups for different form.
-Shows popup when form submitted successfully.
-You can customize popup content, background,font color,button -text,button color,Duration time etc..
-You can customize background as image or add radial gradient colors.
-Popup is fully responsive and compatible in all major browser.
